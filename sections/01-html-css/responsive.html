Welcome to the Responsive page

this page turns blue when the width is less than 600px and red when the width is greater than 600px

HTML Code:

@media (min-width: 600px){
    body{
        background-color: red;
    }
}
@media (max-width: 599px){
    body{
        background-color: blue;
    }
}
